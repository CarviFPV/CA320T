# CA320T — ArduPilot Parameters

This folder contains the ArduPilot `.param` configuration files for the real-life CA320T hardware model.

## Loading Parameters

1. Connect your flight controller to your PC.
2. Open **Mission Planner** or **QGroundControl**.
3. Navigate to **Config / Full Parameter List**.
4. Click **Load from file** and select the `.param` file from this folder.
5. Write the parameters to the flight controller.

> **Hardware files and build instructions** are hosted exclusively on [MakerWorld](https://makerworld.com/).  
> Search for **CA320T by CarviFPV** to find the printable parts and assembly guide.

## Notes

- Always verify parameters after loading before your first flight.
- These parameters are tuned for the stock CA320T build. Modifications to the airframe may require re-tuning.
